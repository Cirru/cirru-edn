
Cirru EDN
----

> EDN data in Cirru format.

### Usage

[![Respo](https://img.shields.io/clojars/v/cirru/edn.svg)](https://clojars.org/cirru/edn)


```edn
[cirru/edn "0.0.1-a1"]
```

```clojure
(cirru-edn.core/parse "a b")

(cirru-edn.core/write {:a 1})
```
